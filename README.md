# 선택 연산자

## 연습 문제

#### [1] 한 개의 정수를 입력 받은 후, 입력받은 수를 출력하고 그 수가 10보다 크면 다음 줄에 “10보다 큰 수를 입력하셨습니다.”라는 메시지를 출력하는 프로그램을 작성하시오.
입력
```
15
```
출력
```
15
10보다 큰 수를 입력하셨습니다.
```

